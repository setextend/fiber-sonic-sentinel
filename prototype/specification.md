## 3. Спецификация системы автономного питания (АКБ)

| Параметр аккумуляторной батареи        | Значение                                         | Техническое обоснование                                                                               |
| :------------------------------------------------------------------ | :------------------------------------------------------- | :-------------------------------------------------------------------------------------------------------------------------- |
| **Тип химического источника тока** | **LiFePO4** (Литий-железо-фосфат) | Срок службы 10–15 лет, стабильная работа при низких температурах         |
| **Конфигурация сборки**                     | **3S2P** (6 ячеек формата 18650)       | Оптимальное рабочее напряжение для DC-DC преобразователей платы         |
| **Номинальное напряжение сборки**  | **9.6 В** (Диапазон: 7.5 В – 10.9 В)   | Безопасный низковольтный контур питания                                                 |
| **Номинальная емкость сборки**        | **3000 мА·ч** (3.0 А·ч)                     | Параллельное удвоение емкости промышленных банок по 1500 мА·ч             |
| **Полная запасенная энергия**          | **28.8 Вт·ч**                                  | Обеспечивает выполнение отраслевого норматива автономности            |
| **Автономность ЛЕТОМ (без ЛЭП)**       | **~13.5 часов**                               | Полное покрытие времени устранения аварии на линии электропередач |
| **Автономность ЗИМОЙ (без ЛЭП)**       | **~7.0 часов**                                | Гарантированное удержание сети в рабочем состоянии зимнюю ночь       |
| **Источник постоянного заряда**      | **Power Harvesting** (Катушка съема)   | Индуктивный разъемный трансформатор тока на проводе ЛЭП (5–10 Вт)       |

## 4. Механические и аэродинамические допуски привода

* **Крутящий момент удержания мотора (с редуктором):** 1.0 Н·м (100 Н·см).
* **Максимальная расчетная нагрузка ветра (ураган 25 м/с):** Сила давления на торец панели — 2.5 Н.
* **Максимальный крутящий момент от ветра на валу:** 0.18 Н·м.
* **Коэффициент запаса мощности привода:** **5.5х** (Обеспечивает гарантированный срез наледи и проворот консоли при любом направлении ветра).
* **Тип климатической защиты оси:** Двухуровневый бесконтактный лабиринтный затвор (вертикальный воротник корпуса высотой 12 мм + перекрывающий защитный зонтик флажка высотой 15 мм с зазором 1.5 мм).
