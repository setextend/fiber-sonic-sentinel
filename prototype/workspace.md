Для этапа подготовки рабочего места и сборки первого лабораторного прототипа (MVP 0.1) на столе вам понадобится стандартный набор инструментов для электроники и программирования (p. 1).

Поскольку на первом этапе сборка идет «на коленке» без пайки (pp. 1, 4), паяльную станцию можно пока не включать, но рабочее пространство нужно организовать правильно.

## 💻 1. Программное обеспечение (на ноутбук/ПК)

Установите этот софт заранее, чтобы не отвлекаться во время сборки:

* VS Code + расширение PlatformIO (или Arduino IDE) — для написания кода под ESP32-S3 (pp. 1, 4).
* Python 3.10+ — понадобится на Этапе 3 для запуска скрипта триангуляции (pp. 1, 5). Сразу установите библиотеки `numpy`, `scipy` и `matplotlib` через терминал (`pip install numpy scipy matplotlib`).
* Audacity — бесплатный аудиоредактор для проверки качества звука со спущенного по Wi-Fi потока (p. 4).
* Драйверы CH340 / CP210x — для того чтобы компьютер «увидел» отладочную плату ESP32 через USB-кабель.

## 🔌 2. Инструменты и тестовое оборудование

* Цифровой мультиметр — самый простой, для прозвонки соединений и проверки напряжения (3.3V и 5V) на макетной плате, чтобы случайно не сжечь микрофоны.
* Качественный кабель USB — Type-C (или Micro-USB, в зависимости от разъема на вашей плате ESP32-S3) (p. 1). Важно: кабель должен быть дата-кабелем (для передачи данных), а не просто шнуром для зарядки смартфона.
* Смартфон — он будет использоваться как генератор звукового сигнала (синусоиды 1–2 кГц) на этапе тестирования пеленгации (p. 5).

## 📦 3. Организация компонентов на столе

Разложите детали из вашего списка (BOM) так, чтобы они были под рукой (p. 1):

* Беспаечная макетная плата (Breadboard) — закрепите её по центру стола (p. 4).
* Провода Dupont (AWG 26-28) — разделите их по цветам (p. 3). *Совет:* используйте красный для питания (3.3V/5V), черный для земли (GND), а цветные (желтый, зеленый, синий) для сигнальных линий I2S/I2C (p. 4). Это спасет от путаницы в 8 микрофонах (pp. 2-3).
* Временная пластина 150х150 мм — вырежьте её из плотного картона, вспененного ПВХ или тонкого пластика, чтобы закрепить первые 4 микрофона по углам, как указано на Этапе 3 инструкции (p. 5). Двусторонний скотч или малярная лента помогут временно зафиксировать микрофоны INMP441 на этой пластине (pp. 2, 5).
